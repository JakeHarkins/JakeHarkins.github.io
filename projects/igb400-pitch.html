Alerian Defenders - Project Pitch

About

Alerian Defenders is a chaotic multiplayer party game where Players take on the role of a crew member aboard the Concordia, a ship headed for the Aleria system. Upon the ship's final approach, a number of systems fail, and the crew must work together to repair the ship before it crashes into the planet. Players will take on different roles aboard the ship, each with their own responsibilities; the problem being that only some Players will be able to see the ship's current status, while others will have to rely on the information provided by their crewmates.

The game is designed to be played in person with a group of friends, or over the web with a similar experience to the Jackbox set of party games. Players will be able to join a lobby hosted on a host device, and play together on their mobile devices.

Project Aims

The aims of this project are to create a multiplayer mobile party game experience, to be played in person with a group or over the web in a similar experience to the Jackbox set of party games.

  • Develop a Mobile Game Experience in Unity
  • Develop a Game using Unity's Netcode for GameObjects Framework

Gameplay Overview

Players will begin the game with one Player hosting a lobby, either on a mobile device or a host device which Players can view together which provides a current overview of the ship. After joining, Players are able to set a nickname, and wait for the host to begin the lobby. For first time Players, the controls and goals of the game are shown, these can be skipped by individuals, or the host of the game.
