Constructing Change

Constructing Change was a project made in collaboration with Awesome Women in Construction (AWIC) in order to encourage young women to join the construction industry.

In Constructing Change, Players control a variety of worksite objects and tools, while exploring an evolving worksite where the new AWIC Headquarters is being built. Players will be able to interact with a variety of workers on the jobsite, which will teach the Player about their role.

The development team feels that we met the brief by engaging Players with an interesting game concept, then using that engagement to teach Players through the world and the dialogue.

Constructing Change was developed in a team consisting of 4 members, myself, Lachlan Taylor (Designer), Mirelle Mimiague (Artist) and Guang Yang (Designer).

Game Features

Dialogue

The Dialogue system is used to let the user interact with NPCs around the game world. The Dialogue is shown letter-by-letter to the Player, improving readability. The Dialogue displays have support for Rich Text Format tags for individual characters, and prevents RTF tags from displaying with the letter-by-letter display.

NPC Randomiser

The NPC Randomiser is used to populate the world with NPCs. There are options for randomisation, or manually setting up the character, which lets us take NPCs through multiple levels. This allows for characters to be preserved from scene to scene, which allows for characters like the Site Manager to lead the Player through each level.

The amount of customisation options available for NPCs is expandable by specifying new colours, hair styles or accessories for the NPC prefab. Adding these to the NPC Prefab before runtime (In Editor) will allow any NPC to use the new customisation options.
